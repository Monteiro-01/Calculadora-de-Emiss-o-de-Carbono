# Calculadora de Emissor de Carbono
# 🌱 Eco Track Go

Um **calculador intuitivo de pegada de carbono** para ajudar você a **medir, acompanhar e reduzir** seu impacto ambiental no dia a dia.  
🔗 Acesse: [Eco Track Go](https://meaty-eco-track-go.base44.app)

---

## 🚀 Funcionalidades

- 📊 **Medição da pegada de carbono**: calcule suas emissões com base em hábitos de transporte, alimentação e consumo.  
- 📈 **Acompanhamento contínuo**: registre e visualize seu histórico para entender sua evolução.  
- 💡 **Sugestões práticas**: receba dicas personalizadas para reduzir seu impacto ambiental.  
- 🎨 **Interface amigável**: design simples e intuitivo para facilitar o uso diário.  

---

## 🛠️ Tecnologias utilizadas

- **Frontend**: React + Vite  
- **Estilização**: TailwindCSS  
- **Backend/API**: Node.js / Express  
- **Deploy**: Base44
